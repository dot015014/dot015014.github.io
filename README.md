# dot015014.github.io

신수진 백엔드 개발자 포트폴리오 초안입니다.

## 수정할 곳

- `index.html`: 이름, 소개 문구, 프로젝트 설명, 이메일 주소
- `styles.css`: 색상, 여백, 반응형 스타일

## GitHub Pages 배포

1. GitHub에서 `dot015014.github.io` 저장소를 public으로 생성합니다.
2. 이 폴더의 파일을 저장소에 push합니다.
3. 저장소의 `Settings > Pages`에서 `Deploy from a branch`를 선택합니다.
4. Branch는 `main`, Folder는 `/root`로 설정합니다.

배포 후 주소는 `https://dot015014.github.io`입니다.

## 검색 노출

현재 `index.html`에는 검색엔진 노출을 줄이기 위해 `noindex, nofollow` 메타 태그가 들어 있습니다.
채용용으로 검색 노출을 허용하고 싶다면 해당 줄을 제거하면 됩니다.
